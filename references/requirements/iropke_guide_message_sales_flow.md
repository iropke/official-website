# IROPKE Sales Narrative Guide (Master Message)

---

## 0. Purpose

This document defines the core sales narrative of IROPKE.

It is not a service description.  
It is a **conversion-driven storytelling structure** designed to:

- create executive-level alignment
- reframe the problem
- eliminate traditional solution assumptions
- position IROPKE as a system provider
- lead naturally to project inquiry

---

## 1. Narrative Flow Structure

```
Problem → Misconception → Real Problem → Insight → System → Solution → Action
```

---

## 2. Opening (Problem Framing)

### Key Message

“Why is our company not appearing in AI?”

### Supporting Context

- Companies have websites  
- Companies produce content  
- Companies invest in marketing  

Yet:

AI does not select them.

---

## 3. Breaking the Misconception

### Key Message

“This is not an SEO problem.”

### Common Misconceptions

- SEO will solve it  
- More content will solve it  
- Website redesign will solve it  

All of these are partial solutions.

---

## 4. Defining the Real Problem

### Core Message

“The problem is not the website.  
It is the digital operating structure.”

### Current State

| Area | Status |
|------|--------|
| Branding | Disconnected |
| Content | Fragmented |
| Search | Reactive |
| Risk | Post-managed |

### Result

Everything operates in isolation.

---

## 5. Key Insight

### Core Message

“AI does not read content.  
It understands structure.”

### Implication

- Structure > Volume  
- Clarity > Keywords  
- System > Output
