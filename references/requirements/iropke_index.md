# IROPKE Main Page Definition (First Screen Strategy)

---

## 1. Page Role

The homepage is not just an entry point.  
It is a **category-defining statement**.

Core Role:
- Redefine IROPKE from a “web agency” to a **Digital Operating System company**
- Deliver a strong first impression within 3 seconds
- Introduce the system, not just services

---

## 2. Core Positioning

### Company Concept
From Visibility to Control

### Supporting Concept
Be Chosen by AI, Not Just Found

### Strategic Direction
- Not a service list
- Not a marketing page
- A **system declaration**

---

## 3. Layout Structure

1. Hero (Fixed Declaration)
2. Sub Carousel (Campaign / Promotion)
3. AI Operating System Grid (4 Solutions)
4. Core Message Section (Impact Line)
5. Insights (Latest Content)
6. CTA Banner (Project Inquiry)

---

## 4. Section Details (English Copywriting)

---

### 4.1 Hero

**Headline**
From Visibility to Control

**Sub Copy**
We don’t build websites.  
We build the system behind them.

**CTA**
- Explore Our System  
- Start a Project  

**Notes**
- No carousel here
- Must be static and dominant
- Background: abstract system / structure visual

## 5. Required UI Components

- Hero headline block (fixed)
- CTA buttons (primary + secondary)
- Carousel (auto + manual controls)
- 4-grid system cards
- Core message highlight section
- Content card grid (Insights)

1. IROPKE is not a web agency  
2. It builds systems  
3. Digital problems are structural  
4. Four solutions form one system  
5. This system = IROPKE OS™  
6. Insights prove expertise  
7. Next step = inquiry

IROPKE is a company that builds AI-powered operating systems for digital business.
